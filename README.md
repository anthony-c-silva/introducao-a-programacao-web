Web App Introduction

## Install the dependencies
```bash
yarn
# or
npm install
```
### Install Quasar Framework Global
```bash
npm i -g @quasar/cli
```
### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```
